# 🎯 QuizMaster – Interactive Quiz Game

An interactive and responsive quiz web application built using **HTML, CSS, and JavaScript**.

Test your knowledge across multiple categories with a timer-based challenge system!

---

## 📌 Features

✨ Multiple Categories  
- 🔬 Science  
- 🏛️ History  
- 🌍 Geography  
- 🎬 Entertainment  

⏳ 15-Second Timer for each question  
📊 Progress bar tracking  
🎯 Score calculation with accuracy percentage  
🌙 Dark / ☀️ Light theme toggle (saved in localStorage)  
⌨️ Keyboard shortcuts (A, B, C, D)  
📱 Fully responsive design  

---

## 🛠️ Built With

- HTML5  
- CSS3 (Modern UI + Theme Variables)  
- JavaScript (Vanilla JS – No frameworks)
